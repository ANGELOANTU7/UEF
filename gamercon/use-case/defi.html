Why Build on Flow

Scalability

Flow’s multi-role architecture is designed to scale with fast transaction finality. Flow can architecturally support up to 1 million TPS and petabytes of user data.

Safety & Security

The built-in security at Flow protocol level. The built-in ownership and resource-oriented model of Cadence make smart contracts safe and secure.

Easy User Onboarding

Flow’s Walletless Onboarding + Hybrid Custody solution simplifies user onboarding.

Complex On-Chain Logic Support

Flow is architecturally designed to scale with on-chain logic complexity and economical to execute complex transactions, such as arbitrage between DEXes, liquidation of a debt position, etc.

Composability

Cadence interface support allows high-level logic to work with a mixture of objects. Scripts & transactions can separate out integration glue from on-chain components; contracts can invoke one another.

Flow Ecosystem

The fast growing Flow ecosystem provides a large user base (20M+ user accounts) & a dedicated community of developers that can inspire and support each other.
